## WORKSTREAM 2: Waterfall Enrichment Suite

### Based on production patterns (leadgenie) + user's reimagined UX

### 2A: Access Email / Access Phone on Search Page

**Current state:** Locked buttons that do nothing.

**Target state:**
1. **Hover states** on "Access email" / "Access phone" explaining enrichment:
   - "Access email will cost 1–4 credits based on your waterfall configuration"
   - Link to "Configure" waterfall settings
2. **Column header icon** — info icon in Email/Phone column headers, also links to configure
3. **Click flow:**
   - Click "Access email" → spinner inline → "Verifying..." text replaces button
   - Completion → show email address inline
4. **Hover on revealed email** — rich popover showing:
   - Verification log (which sources were checked)
   - Credit cost breakdown per source
   - Waterfall visualization (Apollo → Icypeas → LeadMagic → etc.)
   - Final result status (verified/unverified)

### 2B: Bulk Action Credit Preview ("Cart")

**When selecting multiple records + clicking Save/Add to List/Add to Sequence:**
1. Actions panel shifts to a **cart preview** showing:
   - Waterfall settings summary
   - Estimated credit cost for the batch
   - Per-record cost range (min–max)
2. **CTA at bottom** to complete the action
3. **If Save** → just saves
4. **If Add to List** → auto-suggest a list (pick or create)
5. **If Add to Sequence** → auto-suggest a sequence

### 2C: Email Verification Filter

**In SearchPage left sidebar filter list:**
1. Add "Email verification" filter section
2. Inline expandable (not modal) showing:
   - Verified
   - Not verified
   - Waterfall verified
3. Filter badges in the filter list

### 2D: Enrichment Configuration Panel

**Triggered from column header icon or tooltip "Configure" link:**
1. **Right-side panel** (similar to EmailComposeDrawer width — 680px)
2. **Sections:**
   - **Data sources** — draggable ordered list:
     - Apollo (1 credit)
     - Icypeas (1 credit)
     - LeadMagic (1 credit)
     - FindyMail (1 credit)
     - Prospeo (1 credit)
   - Waterfall visualization showing flow: Apollo → Icypeas → LeadMagic → FindyMail → Prospeo
   - **Validation source** — toggle on/off:
     - ZeroBounce (1 credit)
   - **Stop search when:**
     - "A verified email is found" (higher credit usage, single validation)
     - "Any email is found" (lower credit usage, any status)
   - **Credit summary** at top: "1–4 credits per enriched record"
3. **Improved UX vs production:**
   - Visual waterfall diagram (not just a list)
   - Clear credit cost callouts per source
   - Inline explanations of what each source does
   - Live preview of cost impact when reordering
